Introduction and Definition of Trend

Responsive Images have become very popular in Html 5. Using responsive web design (RWD), the images can re-size its self to the screen size. As if the image is set to 100% 100% the image will cover the whole content area, on any screen size, But depending on the picture used it may not look to great. You can use two different size pictures, or use the media query. /* For devices smaller than 400px: */ body { background-image: url('picture...'); } /* For devices 400px and larger: */ @media only screen and (min-device-width: 400px) { body { background-image: url('picture...'); } }
